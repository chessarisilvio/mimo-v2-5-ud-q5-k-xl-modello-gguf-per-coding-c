# C++ Coding Prompts for MiMo-V2.5-UD-Q5_K_XL Testing

## Prompt 1: Basic Hello World
Write a simple C++ program that prints "Hello, World!" to the console.

## Prompt 2: Factorial Function
Implement a recursive function to calculate the factorial of a non-negative integer.

## Prompt 3: Array Sum
Given an array of integers, write a function to return the sum of all elements.

## Prompt 4: Class Definition
Create a simple C++ class named `Rectangle` with private members `width` and `height`, public constructor, and methods to calculate area and perimeter.

## Prompt 5: Sorting Algorithm
Implement the quicksort algorithm to sort an array of integers in ascending order.

## Prompt 6: File I/O
Write a C++ program that reads a text file line by line and prints each line to the console.

## Prompt 7: Pointers and References
Explain the difference between pointers and references in C++ with code examples.

## Prompt 8: Template Function
Create a template function that swaps two values of any type.

## Prompt 9: Inheritance
Create a base class `Shape` with a virtual method `area()`, and derive two classes `Circle` and `Rectangle` from it, overriding the `area()` method.

## Prompt 10: Lambda Expression
Use a lambda expression to sort a vector of strings by their length.
